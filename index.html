AVCaptureMultiCamSession

端侧Apple 原生
多摄像头同时采集 · AVFoundation

  • 调用AVFoundation，Swift/ObjC；通过 AVCaptureConnection 定义数据流向

Core Motion

端侧Apple 原生
运动与设备姿态 · 计步/陀螺仪

Apple 的运动数据框架，读取加速度计、陀螺仪、磁力计与气压计，并提供融合后的设备姿态(CMDeviceMotion)、计步与活动类型(走/跑/骑车/驾车)识别，是健身、AR 稳定、手势交互的底层数据源。

  • 能力计步/距离/楼层、设备姿态(attitude/旋转)、活动识别、原始加速度与陀螺仪
  • 限制CMMotionManager 实时流上限约 100Hz；高频批采(CMBatchedSensorManager 200/800Hz)仅 Apple Watch 较新机型；部分数据需协处理器支持
  • 调用Core Motion(CMMotionManager / CMPedometer / CMMotionActivityManager)
  • 许可系统框架
  • 成本免费(系统框架)
  • 平台iOS / iPadOS / watchOS

Core Location

端云Apple 原生

  • 成本免费(系统框架)

Secure Enclave

端侧Apple 原生
硬件安全子系统 · 密钥底座

集成在 Apple SoC 内、与主处理器隔离的专用安全子系统，独立管理密钥与加解密。私钥在其中生成且永不以明文离开芯片——你只能让它「用」密钥而拿不到密钥本身，主系统内核即便被攻破也保护用户数据。生物识别、Passkeys、App Attest、设备加密等都以它为信任根。

  • 能力片上生成/保管 NIST P-256 密钥，签名/密钥交换/对称加密，独立内存加密(AES-XEX)
  • 限制仅 P-256 曲线；通过上层框架间接调用，非通用加速器
  • 调用经 CryptoKit SecureEnclave / Keychain (kSecAttrTokenIDSecureEnclave) 访问
  • 许可商用闭源
  • 成本免费（系统内置）
  • 平台iOS / macOS / iPadOS / watchOS
